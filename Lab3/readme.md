# Лабораторная работа 3: Перспективная проекция, вращающийся куб

## Описание
Трёхмерное приложение на DirectX 11 с вращающимся кубом и управлением камерой. Куб автоматически вращается вокруг своей оси, камера управляется стрелками клавиатуры.

## Требования задания
- Вывести вращающийся кубик в трёхмерном пространстве
- Добавить управление камерой (вращение вокруг сцены)
- Передавать матрицы model и view-projection в отдельных константных буферах
  
## Управление
- Стрелки влево/вправо — вращение камеры по горизонтали
- Стрелки вверх/вниз — вращение камеры по вертикали
  
## Технологии
- Язык: C++
- IDE: Visual Studio 2022
- API: WinAPI, DirectX 11, DXGI, DirectXMath
- Платформа: x64
- Конфигурации: Debug (отладочный слой) и Release

## Структура кода
- `wWinMain()` — точка входа, создание окна и главный цикл
- `InitDirectX()` — инициализация графики (устройство, цепочка обмена)
- `CreateCubeResources()` — создание буферов вершин и индексов куба
- `CompileShaders()` — компиляция HLSL-шейдеров с двумя константными буферами
- `UpdateCamera()` — обработка ввода с клавиатуры
- `RenderFrame()` — обновление матриц и отрисовка кадра
- `OnResize()` — обработка изменения размера окна
- `CleanupDirectX()` — освобождение ресурсов
