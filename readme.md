# Домашняя работа 1: Инициализация DirectX 11

## Описание
Минимальное графическое приложение на WinAPI с использованием DirectX 11. Окно с тёмно-синим фоном, поддержка изменения размера.

## Требования задания
- Создать оконное графическое приложение
- Проинициализировать DirectX 11
- Заливать всё фиксированным цветом на каждом кадре
- Поддержать изменение размера окна (WM_SIZE + ResizeBuffers)

## Технологии
- Язык: C++
- IDE: Visual Studio 2022
- API: WinAPI, DirectX 11, DXGI
- Платформа: x64
- Конфигурации: Debug (с отладочным слоем) и Release

## Структура кода
- `wWinMain()` — точка входа
- `PeekMessage` цикл — основной цикл приложения
- `InitDirectX()` — инициализация графики
- `OnResize()` — обработка изменения размера
- `RenderFrame()` — отрисовка кадра
- `CleanupDirectX()` — освобождение ресурсов

## Ключевые особенности
- Использован `DXGI_SWAP_EFFECT_FLIP_DISCARD`
- Пропуск "Microsoft Basic Render Driver"
- Проверка всех HRESULT через SUCCEEDED/FAILED
- Отладочный слой DirectX в Debug конфигурации
- Поддержка запуска из VS и отдельно по .exe
